😎 关于我

你好！我是张玉洁（英文名 Prenda）——约翰斯·霍普金斯大学机械工程硕士在读生， 研究方向聚焦于机器人学、计算机视觉与智能控制的交叉领域。

我的学术与科研历程始终围绕解决医疗及仿生机器人领域的复杂问题。 擅长图像处理、运动控制算法设计，以及基于 ROS 的机器人系统集成。 在过往项目中，我曾利用立体视觉与 OpenCV 构建实时深度估计系统， 基于贝塞尔曲线为自主导航开发轨迹规划算法， 并搭建完整的 ROS 传感器融合与可视化流水线。

目前，我正在研究基于光纤布拉格光栅（FBG）的手术工具形状感知课题， 使用 Python 与 ROS、3D Slicer 进行高频数据采集与实时三维可视化控制。 我热衷于将鲁棒控制策略与智能感知相结合，构建能在非结构化环境中可靠运作的系统。

我也热爱开放协作与跨学科问题求解——如果你正在做有趣的事情，欢迎与我联系！💫

📋 科研经历

FBG 导针三维实时形状感知

2024.12 — 2025.07

高级医疗仪器与机器人研究实验室 · 约翰斯·霍普金斯大学

目标：为基于光纤布拉格光栅（FBG）的导针开发实时三维形状感知系统，实现微创手术中无图像的精准针头追踪。

成果：在多次机器人插入实验中 RMSE < 1.3 mm；通过立体视觉对比流水线验证感知精度，支持未来自主针头转向研究。

  • 构建 API 驱动的光谱处理流水线，拟合谱峰并与厂商自动拟合结果交叉验证
  • 将 FBG 输入通道从 4 扩展至 7，并优化基于李群的三维重建算法，在仿体组织实验中精度达 < 2.0 mm
  • 设计对比流水线，通过配准立体相机三维重建与 FBG 感知结果来量化插入形状精度

振动驱动胶囊机器人

2023.01 — 2023.12

深圳仿生机器人与智能系统重点实验室

开发了一种带突出磁性刚毛的振动驱动胶囊机器人（VCR），增强了其在湿/干环境下的适应性，用于胃肠道诊断。 发表于 2023 IEEE ROBIO。

  • 机器人系统设计：利用 SolidWorks 3D 打印并优化磁驱动振动胶囊机器人机械结构
  • 实验测试与运动分析：搭建基于相机的实时跟踪系统，使用 OpenCV 和 MATLAB 分析轨迹、速度与控制精度
  • 功能测试与验证：通过力传感器和示波器测量力与振动特性，基于实时传感器反馈优化系统性能

基于深度感知的履带式结肠内镜机器人

2022.05 — 2024.04

深圳仿生机器人与智能系统重点实验室

利用计算机视觉检测结肠深度，实现履带式结肠内镜机器人的自主导航。 成果发表于 2024 IEEE ICRA， 并获得 专利。

  • 运动规划与控制：设计基于贝塞尔曲线的轨迹优化算法，绕过结肠袋折皱并避免碰撞
  • 机械设计与驱动：使用 SolidWorks 优化蜗轮蜗杆传动机构，提升结肠内推进效率
  • 立体视觉深度感知：利用 OpenCV 和 MATLAB 开发深度估计算法，结肠袋检测精度达 88.11%
  • 实验验证：在离体猪肠中搭建实时运动分析测试平台
